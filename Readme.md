### TreeTagger

Install treetagger following the instructions from http://www.cis.uni-muenchen.de/~schmid/tools/TreeTagger/

```bash
export TREETAGGER_HOME='/path/to/your/TreeTagger/cmd/'
```

### Python libs

pip install -r requirements.txt
